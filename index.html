1. Information We Collect

1.1 Personal Information

  • Account Information: Email address, username, and password when you create an account
  • Profile Information: Display name, profile photo (optional)
  • Authentication Data: OAuth tokens from Google Sign-In and Apple Sign-In (if used)

1.2 Device Information

  • Device Identifiers: Device model, manufacturer, and hardware specifications
  • Platform Information: Operating system version (Android/iOS)
  • Phone State: Read phone state permission (for error reporting only)

1.3 Game Data

  • Match Statistics: Games won, games lost, total games played
  • Ranking Points: Individual and team ranking scores
  • Match History: Records of completed matches (last 20 matches)
  • Team Information: Team memberships, team names, team photos
  • Friend Lists: User IDs of friends you've added
  • Match Requests: Friend match invitations and team invitations

1.4 Usage Data

  • App Activity: Online/offline status, in-match status
  • Error Logs: App crash reports and unexpected errors with optional user feedback
  • Network Status: Internet connectivity status (for online gameplay)

2. How We Use Your Information

2.1 Core Functionality

  • Create and manage your user account
  • Enable online multiplayer gameplay (1v1, 2v2, 4-player modes)
  • Match players for online games
  • Track game progress and scores
  • Maintain global and team leaderboards
  • Store match history and statistics

2.2 Social Features

  • Friend connections and friend requests
  • Team creation and management
  • Team invitations and matchmaking
  • Display player profiles and avatars

2.3 App Improvement

  • Diagnose and fix technical issues
  • Analyze app performance and stability
  • Improve user experience

3. Data Storage and Security

3.1 Firebase Services

We use Google Firebase services to store and manage data:

  • Firebase Authentication: Secure user authentication
  • Cloud Firestore: Real-time database for game data
  • Firebase Storage: Profile photos and team photos

3.2 Security Measures

  • All data transmission is encrypted using HTTPS/SSL
  • User passwords are hashed and never stored in plain text
  • Firebase security rules restrict unauthorized access
  • Multi-factor authentication support via Firebase Auth

3.3 Data Retention

  • Active accounts: Data retained indefinitely while account is active
  • Deleted accounts: All personal data permanently deleted within 30 days
  • Match history: Stored for statistical purposes but anonymized after account deletion
